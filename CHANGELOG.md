# Changelog

## v1.0.0

Initial offline-first release.

- Added timer and manual session logging
- Added session log and analytics
- Added subject, chapter, and subtopic tracking
- Added calendar and countdowns
- Added Day / Week / Month sticky-note To Do board
- Added task priority, completion, single delete, and timeline support
- Added backup, restore, and auto-backup
- Added theme, font, and font color settings
- Added support / feature request and donation link placeholders
- Added roadmap note for `v1.1.x` and `v2.0`
- Added Electron Windows app wrapper
